 Gestionar usuario : Use Case

El admin podrá modificar los datos de los usuarios registrados.

 Gestionar cliente : Use Case

El administrador podrá consultar, modificar o eliminar la información de los clientes registrados.

 Asignar roles : Use Case

Los administradores podrán asignar roles a los usuarios para definir sus permisos.

 Gestionar factura : Use Case

El sistema permitirá registrar una factura incluyendo cliente, usuario, productos, cantidades, precios, subtotales y total general. Se actualizará el stock de cada producto facturado.

 Confirmar pago de factura : Use Case

El sistema permitirá actualizar el estado de una factura a paga al confirmar su pago, bloqueando la posibilidad de futuras modificaciones.

 Gestionar proveedores : Use Case

El sistema permitirá al administrador crear, consultar, modificar proveedores.

 Gestionar producto : Use Case

El usuario autorizado accede al formulario de productos y registra los datos básicos del producto. El sistema valida que el código del producto sea único.

 Gestionar carrito : Use Case

El sistema mostrará los productos añadidos al carrito con opción de modificar cantidades o eliminarlos antes de realizar el pedido.

 Gestionar promociones : Use Case

El sistema permitirá mostrar y registrar promociones incluyendo nombre, porcentaje de descuento y fechas de inicio y fin.

 Gestionar categoria : Use Case

El usuario accede a la sección de categorías desde el panel administrativo para gestionar los nombres de las mismas. El sistema actualizará las relaciones correspondientes con los productos.

 Gestionar movimientos de inventario : Use Case

El sistema debe permitir registrar un nuevo movimiento indicando el producto, tipo de movimiento, cantidad y fecha y consultar el historial de movimientos.
