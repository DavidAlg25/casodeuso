Modificar
Steps
1. El Administrador accede al módulo "Gestionar usuario"
2. SYSTEM muestra la lista de usuarios.
3. El Administrador selecciona el usuario a modificar.
4. SYSTEM muestra los datos actuales del usuario.
5. El Administrador realiza los cambios necesarios.
6. El Administrador confirma la modificación.
7. SYSTEM valida los datos.
8. SYSTEM guarda los cambios.
9. SYSTEM notifica que la modificación fue exitosa.

Eliminar
Steps
1. El Administrador accede al módulo "Gestionar usuario".
2. SYSTEM El muestra la lista de usuarios registrados.
3. El Administrador selecciona un usuario.
4. SYSTEM El muestra la información del usuario seleccionado.
5. El Administrador hace clic en "Eliminar".
6. SYSTEM El solicita confirmación de la eliminación.
7. El Administrador confirma la acción.
8. SYSTEM El valida si el usuario puede ser eliminado
9. SYSTEM El elimina el usuario de la base de datos.
10. SYSTEM El muestra un mensaje de confirmación.

Consultar
Steps
1. El Administrador accede al módulo "Gestión de Usuarios".
2. SYSTEM muestra la lista de usuarios registrados.
3. SYSTEM utiliza filtros de búsqueda (por nombre, rol, documento, etc.) si lo desea.
4. SYSTEM muestra los resultados que cumplen con los criterios de búsqueda.
5. El Administrador selecciona un usuario específico.
6. SYSTEM muestra en pantalla la información detallada del usuario (nombre, correo, rol, estado, etc.).
