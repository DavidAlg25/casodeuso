Actualizar cliente
Steps
1. El Administrador accede al módulo “Gestión de Clientes”.
2. SYSTEM muestra la lista de clientes registrados.
3. El Administrador busca y selecciona el cliente que desea actualizar.
4. SYSTEM muestra los datos actuales del cliente.
5. El Administrador modifica uno o más campos (nombre, dirección, correo, teléfono, etc.).
6. El Administrador guarda los cambios.
7. SYSTEM valida los datos ingresados.
8. SYSTEM actualiza la información del cliente en la base de datos.
9. SYSTEM muestra un mensaje de confirmación.

Eliminar cliente
Steps
1. El Administrador accede al módulo “Gestión de Clientes”.
2. SYSTEM muestra la lista de clientes registrados.
3. El Administrador busca y selecciona el cliente que desea eliminar.
4. SYSTEM muestra los datos del cliente seleccionado.
5. El Administrador hace clic en la opción “Eliminar”.
6. SYSTEM solicita confirmación.
7. El Administrador confirma la eliminación.
8. SYSTEM valida si el cliente puede ser eliminado (por ejemplo, que no tenga facturas o pedidos activos).
9. SYSTEM elimina el registro del cliente.
10. SYSTEM muestra un mensaje de confirmación de eliminación.

Consultar cliente
Steps
1. El Administrador accede al módulo “Gestión de Clientes”.
2. SYSTEM muestra la lista de clientes registrados.
3. El Administrador utiliza filtros o búsqueda para encontrar un cliente específico (por nombre, número de documento, correo, etc.).
4. SYSTEM muestra los resultados que coinciden con los criterios ingresados.
5. El Administrador selecciona un cliente de la lista.
6. SYSTEM muestra en pantalla la información detallada del cliente (nombre, dirección, contacto, historial de compras, etc.).
