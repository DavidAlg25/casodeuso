Inicio de Sesión Exitoso
Steps
1. El Usuario accede a la página de inicio de sesión.
2. SYSTEM muestra el formulario de login.
3. El Usuario ingresa sus credenciales.
4. SYSTEM valida el formato de los datos.
5. SYSTEM busca las credenciales en la base de datos.
6. SYSTEM verifica que el usuario exista y la contraseña sea correcta.
7. SYSTEM El sistema valida que la cuenta esté activa.
8. SYSTEM autentica al usuario.
9. SYSTEM redirige al usuario a su panel de inicio o dashboard.

Recuperación de Contraseña
Steps
1. El Usuario accede a la opción "¿Olvidaste tu contraseña?" desde la página de inicio de sesión.
2. SYSTEM muestra un formulario para ingresar el correo electrónico asociado a la cuenta.
3. El Usuario ingresa su correo electrónico y envía la solicitud.
4. SYSTEM verifica que el correo esté registrado.
5. SYSTEM genera un token o enlace único de recuperación.
6. SYSTEM envía un correo electrónico al usuario con el enlace para restablecer la contraseña.
7. El Usuario accede al enlace recibido en su correo.
8. SYSTEM valida el token y muestra un formulario para ingresar una nueva contraseña.
9. El Usuario ingresa y confirma su nueva contraseña.
10. SYSTEM valida la contraseña y la actualiza en la base de datos.
11. SYSTEM muestra un mensaje de confirmación: “Tu contraseña ha sido restablecida exitosamente”.
12. El Usuario es redirigido a la página de inicio de sesión.
