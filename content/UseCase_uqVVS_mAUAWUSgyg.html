Asignar rol a un nuevo usuario
Steps
1. El Administrador accede al módulo de gestión de roles.
2. SYSTEM muestra la lista de usuarios sin rol asignado.
3. El Administrador selecciona un usuario.
4. SYSTEM muestra los datos básicos del usuario.
5. El Administrador selecciona un rol del listado disponible (por ejemplo: Cliente, Empleado, Administrador).
6. El Administrador confirma la asignación.
7. SYSTEM valida la información.
8. SYSTEM asigna el rol al usuario seleccionado.
9. SYSTEM muestra un mensaje de confirmación de asignación exitosa.

Modificar el rol de un usuario existente
Steps
1. El Administrador accede al módulo de gestión de roles.
2. SYSTEM muestra la lista de usuarios con roles asignados.
3. El Administrador busca y selecciona el usuario cuyo rol desea modificar.
4. SYSTEM muestra el rol actual del usuario.
5. El Administrador selecciona un nuevo rol del listado disponible.
6. El Administrador confirma la modificación.
7. SYSTEM valida la información (que el nuevo rol sea válido y diferente del actual).
8. SYSTEM actualiza el rol del usuario en la base de datos.
9. SYSTEM muestra un mensaje de confirmación de que el rol fue actualizado exitosamente.

Eliminar un rol asignado a un usuario
Steps
1. El Administrador accede al módulo de gestión de roles.
2. SYSTEM muestra la lista de usuarios con roles asignados.
3. El Administrador busca y selecciona al usuario deseado.
4. SYSTEM muestra los roles asignados actualmente al usuario.
5. El Administrador selecciona el rol que desea eliminar.
6. El Administrador confirma la acción de eliminación.
7. SYSTEM valida que se pueda eliminar el rol (por ejemplo, que no sea el único rol que habilita acceso al sistema o que el usuario no esté en sesión activa).
8. SYSTEM elimina el rol del usuario.
9. SYSTEM muestra un mensaje de confirmación.

Consultar rol de un usuario
Steps
1. El Administrador accede al módulo de gestión de roles.
2. SYSTEM muestra un buscador o listado de usuarios registrados.
3. El Administrador busca y selecciona al usuario cuyo rol desea consultar.
4. SYSTEM muestra los datos básicos del usuario.
5. SYSTEM muestra los roles actualmente asignados al usuario.
6. El Administrador cierra la consulta o retorna al menú anterior.
