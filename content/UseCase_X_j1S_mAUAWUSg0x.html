Registrar entrada de productos al inventario
Steps
1. El Usuario autorizado accede al módulo de movimientos de inventario.
2. SYSTEM muestra las opciones disponibles (entrada, salida, consulta).
3. El Usuario autorizado selecciona “Registrar entrada”.
4. SYSTEM muestra un formulario para registrar una nueva entrada.
5. El Usuario autorizado selecciona el producto o productos a ingresar.
6. El Usuario autorizado ingresa la cantidad, fecha, motivo, y otros datos requeridos.
7. SYSTEM valida la información ingresada.
8. SYSTEM registra el movimiento de entrada en la base de datos.
9. SYSTEM incrementa el stock del producto correspondiente.
10. SYSTEM muestra un mensaje de confirmación: “Entrada registrada exitosamente”.

Registrar salida de productos del inventario
Steps
1. El Usuario autorizado accede al módulo de movimientos de inventario.
2. SYSTEM muestra las opciones disponibles (entrada, salida, consulta).
3. El Usuario autorizado selecciona “Registrar salida”.
4. El sistema muestra un formulario para registrar una nueva salida.
5. El Usuario autorizado selecciona el producto o productos a retirar del inventario.
6. El Usuario autorizado ingresa la cantidad, fecha, motivo (ej. venta, pérdida), y otros datos requeridos.
7. SYSTEM valida la información y verifica que haya suficiente stock disponible.
8. SYSTEM registra el movimiento de salida en la base de datos.
9. SYSTEM descuenta la cantidad correspondiente del inventario.
10. SYSTEM muestra un mensaje de confirmación: “Salida registrada exitosamente”.

Modificar un movimiento de inventario
Steps
1. El Usuario autorizado accede al módulo de movimientos de inventario.
2. SYSTEM muestra la lista de movimientos registrados.
3. El Usuario autorizadobusca y selecciona el movimiento que desea modificar.
4. SYSTEM muestra los detalles del movimiento.
5. El Usuario autorizado edita uno o más campos permitidos (por ejemplo, cantidad, motivo o producto).
6. SYSTEM valida los nuevos datos ingresados.
7. SYSTEM actualiza el movimiento en la base de datos.
8. SYSTEM Si la cantidad fue modificada, ajusta el stock en consecuencia.
9. SYSTEM registra un log o auditoría del cambio realizado (si aplica).
10. SYSTEM muestra un mensaje de confirmación: “Movimiento actualizado correctamente”.

Eliminar un movimiento de inventario
Steps
1. El Usuario autorizado accede al módulo de movimientos de inventario.
2. SYSTEM muestra la lista de movimientos registrados.
3. El Usuario autorizado filtra o busca el movimiento que desea eliminar.
4. SYSTEM muestra los detalles del movimiento.
5. El Usuario autorizado hace clic en “Eliminar movimiento”.
6. SYSTEM solicita confirmación de la acción.
7. El Usuario autorizado confirma la eliminación.
8. SYSTEM elimina el registro del movimiento.
9. SYSTEM ajusta automáticamente el inventario si corresponde (revierte la entrada o salida).
10. SYSTEM registra la acción en el historial de auditoría.
11. SYSTEM muestra un mensaje: “Movimiento eliminado correctamente”.
