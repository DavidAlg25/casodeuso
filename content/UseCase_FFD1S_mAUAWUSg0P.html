Crear promoción de producto
Steps
1. El Usuario autorizado accede al módulo de promociones.
2. SYSTEM muestra las promociones activas y la opción “Crear nueva promoción”.
3. El Usuario autorizado selecciona “Crear nueva promoción”.
4. SYSTEM muestra un formulario para ingresar los datos de la promoción
5. El Usuario autorizado completa el formulario y confirma la creación.
6. SYSTEM valida los datos ingresados.
7. SYSTEM guarda la promoción en la base de datos.
8. SYSTEM activa la promoción en las fechas programadas.
9. SYSTEM muestra un mensaje: “Promoción creada exitosamente”.
