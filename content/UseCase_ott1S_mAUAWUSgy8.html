Emitir factura de venta
Steps
1. El Usuario autorizado finaliza el proceso de venta desde el módulo de ventas o caja.
2. SYSTEM recopila los datos de la venta: cliente, productos, cantidades, precios, descuentos e impuestos.
3. SYSTEM genera automáticamente el número de factura.
4. SYSTEM calcula el total de la venta, incluyendo impuestos.
5. SYSTEM genera el documento de factura (formato digital y/o impreso).
6. SYSTEM guarda la factura en la base de datos.
7. SYSTEM vincula la factura con el cliente y la venta correspondiente.
8. SYSTEM muestra un mensaje: “Factura generada correctamente”.
9. SYSTEM ofrece al empleado o cliente opciones para imprimir o enviar la factura por correo.
