Use Case - Gestionar categoria link

Properties
Name Value
Description

El usuario accede a la sección de categorías desde el panel administrativo para gestionar los nombres de las mismas. El sistema actualizará las relaciones correspondientes con los productos.

Registrar categoria
Steps
1. El Usuario autorizado accede al modulo "Gestionar categoria"
2. SYSTEM muestra la lista de categorias existentes
3. El Usuario autorizado hace clic en "Registrar categoria"
4. SYSTEM muestra el formulario de registro
5. El Usuario autorizado ingresa los datos de la categoria (nombre, descripcion, etc)
6. El Usuario autorizado hace clic en "Guardar"
7. SYSTEM valida los datos ingresados
8. SYSTEM registra la categoria en la base de datos
9. SYSTEM muestra un mensaje de confirmacion y actualiza la lista

Actualizar categoria
Steps
1. El Usuario autorizado accede al modulo "Gestionar categoria"
2. SYSTEM muestra la lista de categorias existentes
3. El Usuario autorizado selecciona una categoria
4. SYSTEM muestra los datos actuales de la categoria
5. El Usuario autorizado modifica uno o mas campos
6. El Usuario autorizado hace clic en "Actualizar"
7. SYSTEM valida los datos modificados
8. SYSTEM actualiza la informacion en la base de datos
9. SYSTEM muestra un mensaje de confirmacion

Eliminar categoria
Steps
1. El Usuario autorizado accede al modulo "Gestionar categoria"
2. SYSTEM muestra la lista de categorias existentes
3. El Usuario autorizado selecciona una categoria
4. El Usuario autorizado hace clic en "Eliminar"
5. SYSTEM solicita informacion
6. El Usuario autorizado confirma la eliminacion
7. SYSTEM elimina la categoria de la base de datos
8. SYSTEM muestra un mensaje de confirmacion y actualiza la lista

Consultar categoria
Steps
1. El Usuario autorizado accede al modulo "Gestionar categoria"
2. SYSTEM muestra la lista de categorias existentes
3. El Usuario autorizado ingresa un criterio de busqueda (nombre, palabra clave, etc)
4. El Usuario autorizado hace clic en "Buscar"
5. SYSTEM filtra y muestra las categorias que coinciden con el criterio
